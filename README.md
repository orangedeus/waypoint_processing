# Waypoint Processing

## Requirements and Dependencies

- Anaconda v4.9.2
- FFMPEG v4.3.2
- Python v3.8.5
- Node v15.11.0
- ExpressJS

## Installation

- Run `npm install` in cloned directory
- Configure `config.js` and save and edit `example.env.local` as `.env.local` with needed values.
- Use `npm start` to run
